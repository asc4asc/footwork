# Klopfen

#### Material:

Keins (Start Linie)

#### Kurzbeschreibung:

Start durch Klopfen an eine Linie Fechtschritte vorwärz. Doppel klopfen Fechtschritte zurück zur Linie.

Start durch Klopfen an eine Linie Fechtschritte vorwärz. Klopfen beschleunigen der Fechtschritte. Doppel klopfen Fechtschritte zurück zur Linie.

Start durch Klopfen an eine Linie Fechtschritte vorwärz. Klopfen beschleunigen der Fechtschritte. Klopfen z.B. Ausfall oder Schritt vor Ausfall .... Doppel klopfen Fechtschritte zurück zur Linie.

Nach doppelt Klopfen neuer Start durch einfaches Klopfen.

#### Ziel:
Konzentration, Kardio, Fechtbewegungen lernen.

#### Variationen
Zusätzlich Spiegel oder spezielle Bewegung durch optisches Signal z.B. Hand oben.
